HTML HTML5
Limited number of semantic elements available, such as <header>, <footer>, <nav>, etc. Introduces a comprehensive set of semantic elements (<section>, <article>, <aside>, etc.) for clearer document structure.
Limited multimedia support; reliance on third-party plugins like Flash for audio and video. Native support for <audio> and <video> elements, enabling embedded multimedia without plugins. Also introduces the <canvas> element for dynamic graphics and animations.

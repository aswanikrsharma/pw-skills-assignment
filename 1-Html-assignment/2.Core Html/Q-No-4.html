1. Heading tag(h1,h2,h3,h4,h5,h6)
    • Headings are used to define the titles or subtitles of a web page or a section of content. Example - <h1></h1>
  2. Paragraphs tag
    • The HTML <p> element defines a paragraph.
  3. Horizontal line tag
    • The <hr> tag defines a thematic break in an HTML page, and is most often displayed as a horizontal line.
  4. Span tag
    • The <span> tag functions as an inline container, allowing us to markup and highlight specific portions of text within a document. It serves to emphasise or apply formatting to those particular sections without affecting the overall layout or structure of the page.
  5. Doctype tag
    • We start the file with <!DOCTYPE html>. It tells the browser what type of document to expect.
  6. Head tag (<head></head>)
    • This is the head section of the HTML document, which contains meta-information about the document, such as the title of the web page that appears in the browser's title bar or tab.
  7. Title tag <title>
    • This is the title tag, which sets the title of the web page.
